# How to set up parent control on the Cudy router?

Parent control is used to control the internet access time and websites of certain devices, such as the phone of the kid, and at the same time it will allow other devices to have free access.

Take WR3000 as an example to introduce how to set up the parental control function. The setting requirements are as follows:    
<li> The phone of the kid can only use the Internet from 8:00 am to 10:00 pm on weekends. </li>
<li> Prevent kid from accessing designated websites, such as <i>www.facebook.com</i> and <i>www.youtube.com</i>. </li>


1. Please login to the web page of WR3000 by [http://cudy.net/](http://cudy.net/) or [http://192.168.10.1/](http://192.168.10.1/). Input the password to login.

2. Check the Hostname and MAC address of the phone on the clients list. Go to **System Status** page, Click **More Details** on the **Devices** card, as shown below.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/devices_card.webp" alt="devices" width="800px" style="display: block; margin: 0 auto;">

    Go to the **Devices** page and find the phone of the kid, such as the **Xiaomi-13** on the page. We assume this is the phone of the kid.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/devices_list.webp" alt="devices" width="800px" style="display: block; margin: 0 auto;">

3. Go to **Parental Control**, click on **Add** to create a new **Profile**. Input the Profile Name, such as **Kids' phone**. Click **Save & Apply**.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/create_profile.webp" alt="create" width="800px" style="display: block; margin: 0 auto;">

    The profile is created.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/create_profile_success.webp" alt="create" width="800px" style="display: block; margin: 0 auto;">

    Click on **Add** to choose the phone. Click **Save**.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/add_devices.webp" alt="add" width="800px" style="display: block; margin: 0 auto;">

    The phone is listed here. Click **Save & Apply**.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/add_devices_success.webp" alt="add" width="800px" style="display: block; margin: 0 auto;">

4. Go to **Online Time** page, click **Add** to set the online time schedule **08:00 am--10:00 pm** and choose the **Saturday** and **Sunday**. Click **Save**.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/online_time_a.webp" alt="online" width="800px" style="display: block; margin: 0 auto;">

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/online_time_b.webp" alt="online" width="600px" style="display: block; margin: 0 auto;">

    Click **Save & Apply**. The online time configuration finished.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/online_time_c.webp" alt="online" width="800px" style="display: block; margin: 0 auto;">

5. Go to **Web Filter** page, set the **Internet Access** to **Blacklist**. Add the domain key words **facebook** and **youtube** to the Blacklist. Click **Save & Apply**.

    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/faq/10004/web_filter.webp" alt="web" width="600px" style="display: block; margin: 0 auto;">

So now the phone under the Parental Control list can only access the internet from 8:00 am to 10:00 pm on weekends, and cannot access websites [www.facebook.com](https://www.facebook.com) and [www.youtube.com](https://www.youtube.com). Other devices not in the list have free internet access all the time.
